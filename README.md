# 매칭미? (Matching me?)

> 남도학숙생을 위한 룸메이트 매칭 플랫폼

**매칭미?**는 남도학숙(동작관·은평관)에서 생활하는 재학생들이, 라이프스타일에 맞는 룸메이트를 쉽게 찾을 수 있도록 돕는 서비스입니다. Google 로그인으로 가입하고, 생활 패턴·청결도·흡연 여부 등을 입력하면 AI가 궁합이 좋은 학숙생을 추천해 드려요.

---

## 📱 서비스 소개

- **대상**  
  남도학숙 재사생(동작관·은평관 입주 예정/입주 중인 학숙생)

- **목적**  
  같은 기숙사 내에서 생활 패턴이 비슷한 룸메이트 후보를 탐색하고, 마음에 드는 분과 연락할 수 있게 하는 것

- **특징**  
  - 동작관/은평관 구분으로 같은 관 학숙생만 노출  
  - 아침형·밤형, 청결도, 소음 민감도, 흡연 여부 등으로 필터링  
  - AI가 두 사람의 라이프스타일을 비교해 매칭 포인트 요약 제공  
  - 찜하기로 관심 있는 프로필 저장, 연락처는 하루 3회까지 공개 요청 가능  

---

## 📖 사용 방법

### 1. 로그인

- 서비스에 접속한 뒤 **「Google로 계속하기」** 버튼을 눌러 Google 계정으로 로그인합니다.
- 처음 이용 시 자동으로 가입되며, 이용약관·개인정보 처리방침에 동의한 것으로 간주됩니다.

### 2. 프로필 작성 (온보딩)

- 로그인 후 **3단계**에 걸쳐 본인 정보를 입력합니다.
  - **1단계**: 이름, 성별, 계열, 학년, 소속관(동작관/은평관)
  - **2단계**: 연락처(카카오톡 ID 등)
  - **3단계**: 생활 패턴(아침형/밤형), 잠버릇, 흡연 여부, 청결도, 소음 민감도, 자기소개
- **「프로필 완성하기」**를 누르면 저장 후 대시보드로 이동합니다.

### 3. 대시보드에서 룸메이트 탐색

- **전체 매칭** 탭: 같은 관의 룸메이트 공고 목록을 볼 수 있습니다.
- **AI 추천** 탭: 나와 매칭률이 높은(75% 이상) 학숙생을 우선 볼 수 있습니다.
- 각 카드에는 **이름, 계열, 학년, 성별**, 생활 태그(아침형/밤형, 흡연 여부 등), 자기소개가 표시됩니다.
- **필터**로 관·성별·계열·학년·흡연 여부를 조합해 검색할 수 있습니다.

### 4. 프로필 상세 보기

- 카드를 탭하면 상대방의 **상세 프로필**을 볼 수 있습니다.
- 생활 패턴, 청결도, 소음 민감도, 자기소개와 함께, AI가 생성한 **두 사람의 매칭 포인트 요약**을 확인할 수 있습니다.

### 5. 찜하기 & 연락처 공개

- 마음에 드는 프로필은 **찜하기** 버튼으로 저장해 두었다가, **프로필 > 찜한 룸메이트**에서 다시 볼 수 있습니다.
- **연락처 공개**를 요청하면 카카오톡 ID·기타 연락처를 확인할 수 있으며, **하루 3회**까지 공개 요청이 가능합니다. (초과 시 다음 날 자정에 초기화)

### 6. 마이페이지·설정

- **프로필**: 내 정보 조회·수정, 찜한 룸메이트 목록으로 이동할 수 있습니다.
- **설정**: 계정(로그아웃, 회원 탈퇴), 서비스 소개, 이용약관·개인정보 처리방침, 도움말·지원 링크를 제공합니다.
- 앱 내 **서비스 소개** 페이지에서는 매칭미?의 이용 대상·기능·보안을 한눈에 볼 수 있습니다.

## 🚀 개발자용: 로컬 실행

### 요구사항

- Node.js 18.x 이상
- pnpm(권장) 또는 npm/yarn
- Supabase 프로젝트, Google OAuth 클라이언트 ID

### 설치·실행

```bash
git clone <repository-url>
cd matching-me
pnpm install
```

`.env.local`에 다음을 설정한 뒤:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속. 로그인 시 `/dashboard`로 이동합니다.
